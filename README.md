# 🛒 Retail Grocery Sales Analytics Dashboard

An end-to-end Retail Grocery Sales Analytics project built using SQL, , ETL, and Power BI.

## Project Overview

This project analyzes grocery store sales data to uncover business insights related to revenue, customer purchasing behavior, product performance, payment methods, and sales trends.

The solution includes:

- SQL database design
- Automated data generation
- ETL pipeline
- Data modeling
- Interactive Power BI dashboard
- Business KPIs
- Executive reporting

## Business Insights

- Beverage products generated the highest revenue.
- Weekend sales were significantly higher than weekdays.
- UPI accounted for the largest share of payments.
- Premium customers contributed a substantial portion of revenue.
- Sales peaked during evening hours.
