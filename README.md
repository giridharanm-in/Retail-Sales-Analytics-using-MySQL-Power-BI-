# 🛒 Retail Grocery Sales Analytics Dashboard

> An end-to-end Business Intelligence project built using **MySQL, Python, SQL, and Power BI** to analyze grocery store sales, customer behavior, product performance, and revenue trends.

---

## 📌 Project Overview

The Retail Grocery Sales Analytics Dashboard provides interactive insights into sales performance, customer purchasing patterns, product profitability, and payment trends. The project demonstrates the complete analytics workflow—from data storage and transformation to visualization and business reporting.

This project was developed to simulate a real-world retail analytics solution used by business managers and decision-makers.

---

## 🚀 Features

- Executive Sales Dashboard
- Category Performance Analysis
- Customer Insights Dashboard
- Interactive slicers and filters
- KPI tracking
- Dynamic DAX Measures
- Professional dashboard design
- Star Schema Data Model
